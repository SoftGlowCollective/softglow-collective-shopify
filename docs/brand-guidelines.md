# Brand guidelines — SoftGlow Collective

This document provides the core visual tokens and simple usage guidance for SoftGlow Collective. Use these guidelines as the single source of truth for colors, typography, logo usage, and imagery to keep the brand cohesive across the store, ads, and emails.

Color palette

- Blush Pink: #F6D7D9 (accent, CTA backgrounds, soft highlights)
- Cream White: #FFF8F2 (primary background)
- Soft Beige: #EFE7DA (panels, subtle backgrounds)
- Warm Gold: #CFA56A (accent, CTAs, decorative elements)
- Neutral text: #111111 (primary copy)

Typography

- Headings: Playfair Display or similar modern serif (elegant, high-contrast headings)
- Body: Inter, System UI, or Roboto (clean sans-serif for legibility)
- Scale: H1 40-48px, H2 28-34px, H3 20-24px, body 16px
- Line height: 1.3	61.6 depending on display

Logo

- Provide versions: full logo (wordmark + mark), wordmark only, mark only (square)
- Use transparent PNGs and SVGs when available. Prefer SVG for web use.
- Clearspace: keep at least the height of the logo around it on all sides.
- Minimum size: ensure legibility on small devices (e.g., 40px height for mark)

Imagery & Photography

- Tone: soft, natural light, minimal backgrounds, warm highlights
- Subjects: smiling faces, close-ups of skin textures, product-in-hand shots
- Styling: clean props, neutral backgrounds that complement the brand colors
- File types: use optimized WebP for web delivery, provide fallbacks for older browsers

UI guidance

- Buttons: primary CTA use Warm Gold or blush depending on context. Ensure contrast for accessibility.
- Trust badges: place near Add to Cart and checkout prompts.
- Spacing: generous white space to create a premium, calm feel.

Assets

- Keep original high-res files in `docs/design-assets/` and export optimized WebP/JPEG/PNG for the store.

Notes

- These tokens are a starting point. For large campaigns or offline materials, consult a designer to expand the system.
